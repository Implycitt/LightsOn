## Getting Started


### Prerequisites

* Python/pip
  ```sh
  py -m pip install --upgrade pip
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Implycitt/LightsOn.git
   ```
2. Install Python packages
   ```sh
   pip install PyQt5 numpy scipy functools 
   ```
3. Run
  ```sh
  cd src/
  python main.py 
  ```
